# Ionic-specific Guidelines

These guidelines are specific to Ionic projects. See [AGENTS.md](./AGENTS.md) for general guidelines.

---

## Ionic Usage

- Generate components, pages, and services with the **@ionic/cli**.
- Remove empty constructors and unused lifecycle hooks.
- Follow **Ionic docs or project examples** for correct component usage.
- When using **Ionicons**, import them via `addIcons` in the component constructor.
